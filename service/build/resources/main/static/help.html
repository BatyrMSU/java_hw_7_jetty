Справка по работе с веб-сервисом

Для получения информации по товарам необходимо перейти по адресу: http://localhost:3466/products

Для добавления нового товара необходимо ввести параметры name и amount по тому же адресу, используя метод POST.

Добавление нового товара разрешено только роли manger, просмотр всех товаров доступен ролям manager и guest.
